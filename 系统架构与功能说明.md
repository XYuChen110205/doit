# DoIt 待办管理系统 - 架构与功能说明

## 一、系统概览

**产品定位**：多角色任务管理系统，支持多端同步  
**目标用户**：学生、职场人士、自由职业者  
**核心特色**：角色模式切换、多端数据同步、丰富的任务管理

---

## 六、设计建议

### 当前风格
- **主色调**：水蓝色 (#4A9FD4)
- **背景色**：浅灰蓝 (#F5F9FB)
- **字体**：Noto Sans SC（思源黑体）
- **风格**：清爽、简洁、留白充足

### 建议优化方向
1. **视觉层次**：加强卡片阴影和圆角区分
2. **动效**：添加页面切换和交互动画
3. **空状态**：设计更友好的空状态插图
4. **响应式**：优化移动端触控区域
5. **暗色模式**：增加深色主题
6. **插画**：为不同角色设计专属插图

## 七、技术栈

- **前端**：Vue 3 + TypeScript + Vite
- **后端**：FastAPI + SQLAlchemy
- **数据库**：SQLite (本地) / PostgreSQL (云端)
- **部署**：Cloudflare Pages (前端) + Vercel (后端)
- **字体**：Google Fonts (Noto Sans SC)
